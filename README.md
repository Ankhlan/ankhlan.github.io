# Journal — Old Book

This is a minimal writing blog for GitHub Pages (Option A — plain HTML + Markdown posts).

Structure:

- `index.html` — home page
- `posts/` — Markdown posts (rendered manually or via simple script)
- `assets/css/styles.css` — styles

To publish:
- If this repo is named `username.github.io`, push to `main` and GitHub Pages will serve the `index.html`.
- For project pages, configure GitHub Pages to use the `gh-pages` branch or `docs/` folder.

Optional: add a small build script to convert `posts/*.md` to `posts/*.html` if you want Markdown converted automatically.
