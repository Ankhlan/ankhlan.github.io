Connect to GitHub

Generate a fine-grained personal access token, paste it below, and the desk can save edits directly to Ankhlan/ankhlan.github.io from any browser.

  1. Open github.com/settings/personal-access-tokens/new.
  2. Resource owner: Ankhlan. Repository access: Only select repositories → ankhlan.github.io.
  3. Permissions: Contents → Read and write. Everything else read-only or off.
  4. Set an expiration that suits you. Generate, copy the token, paste below.

Stored only in this browser's localStorage. Never sent anywhere except api.github.com.
