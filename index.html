This site is its own proof of concept. It is published by a small Python pipeline that compiles each post from a single markdown file containing both prose and embedded TeX. Mathematics and figures are rendered by real TeX (compiled to SVG via dvisvgm) and embedded as islands inside the HTML body; the prose itself flows as ordinary HTML so the browser can do what browsers do well — reflow, search, accessibility, mobile.

The result is a real-time writing surface: the author edits one file; a watcher recompiles only what changed; the page reloads with TeX-quality math next to web-native prose. The site you are reading was built by exactly that loop.
